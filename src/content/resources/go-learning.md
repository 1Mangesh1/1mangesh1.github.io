A curated collection of the best resources for learning Go, from your first "Hello, World" to production-ready applications.

## Official Resources

- **[A Tour of Go](https://go.dev/tour/)** — The official interactive tutorial. Start here. It covers everything from basics to concurrency in a hands-on browser environment.
- **[Effective Go](https://go.dev/doc/effective_go)** — The definitive guide to writing idiomatic Go. Read this once, then read it again after you've written your first project.

## Learn by Example

- **[Go by Example](https://gobyexample.com/)** — Annotated example programs covering 70+ Go concepts. Perfect for quick reference and learning patterns.
- **[Exercism Go Track](https://exercism.org/tracks/go)** — Practice problems with mentor feedback. Great for building muscle memory with Go idioms.

## Books

- **Learning Go** by Jon Bodner — The best modern Go book for experienced developers coming from other languages. Covers modules, generics, and current best practices.
- **Let's Go** and **Let's Go Further** by Alex Edwards — Hands-on project-based books that build a complete web application. Excellent for learning Go in a practical context.
- **The Go Programming Language** by Donovan & Kernighan — The classic deep dive. Dense but thorough.

## Video & Courses

- **[JustForFunc](https://www.youtube.com/c/JustForFunc)** by Francesc Campoy — Practical Go screencasts covering real-world topics.
- **[Go Class by Matt Holiday](https://www.youtube.com/playlist?list=PLoILbKo9rG3skRCj37Kn5Zj803hhiuRK6)** — Comprehensive free YouTube course.
